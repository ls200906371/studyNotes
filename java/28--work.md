# 电子商务系统
	* 分布式，集群，负载均衡，高可用
	* 海量数据，业务复杂 
	* 分布式：把系统按照模块拆分成多个子系统，系统之间使用远程通信 

# SOA ： service Oriented Architecture 面向服务的架构
	* 把工程拆分为服务层、表现层两个工程，不同模块之间的服务层和表现层可以相互调用，解决通用业务的问题
